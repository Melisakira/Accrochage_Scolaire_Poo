# Accrochage Scolaire — Projet POO

Application console C# de suivi et de prévention du décrochage scolaire.

## Domaine

Outil de gestion permettant à une équipe éducative de suivre les élèves à risque, d'enregistrer les évènements significatifs (absences, retards, résultats, signalements), de calculer un score de risque selon une logique tenant compte de la temporalité, de la progression et de la combinaison des événements, et de déclencher des alertes automatiques vers les intervenants concernés.

## Stack technique

- Langage : C#
- Type : Application console
- Package UI : Spectre.Console
- IDE : Visual studio 2022

## Lancer le projet

Ouvrir le fichier .sln dans Visual Studio 2022
Appuyer sur F5 ou cliquer sur Démarrer

## Structure du repo

- src/ → code source C#
- uml/ → diagrammes de classes
- docs/ → conception et justifications

## Contexte et motivation 

En Fédération Wallonie-Bruxelles, les acteurs de la lutte contre le décrochage scolaire sont nombreux mais morcelés - chaque service développe ses propres pratiques et outils, sans vision partagée. Sur le terrain, les intervenants ne se connaissent pas toujours, les actions se superposent ou ne se coordonnent pas, et le jeune se retrouve souvent sans suivi cohérent.
Cette application ne remplace pas les acteurs existants — elle propose un outil simple, sans IA, centré sur l'équipe éducative de proximité, pour centraliser le suivi, coordonner les alertes et garder l'humain au cœur des décisions.
Dans sa version actuelle, l'application est un prototype console développé dans le cadre d'un cours de POO.
